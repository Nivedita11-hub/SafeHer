# 🚨 SafeHer – Discreet Safety Assistant

SafeHer is a smart personal safety application designed to help individuals in dangerous situations request help without attracting attention.

The application appears as a normal calculator but contains hidden emergency features that can be triggered using a secret code or gesture.

---

## 💡 Concept
SafeHer blends into everyday use while providing powerful emergency tools in critical situations.

---

## 🔐 Key Features
- 🔑 Hidden SOS trigger using secret code
- 📩 Emergency SMS alert to trusted contact
- 📍 Live GPS location sharing (Google Maps link)
- 🎙 Automatic audio recording for evidence
- 📸 Webcam photo capture during emergency
- 🕒 Incident logging with timestamps

---

## 🛠 Technologies Used
- Python
- Tkinter (GUI)
- OpenCV
- Geolocation API
- Twilio API

---

## ⚙️ How It Works
1. App opens as a normal calculator
2. Enter secret code → opens hidden dashboard
3. Trigger SOS manually or via gesture
4. App:
   - Captures location
   - Records audio
   - Takes photo
   - Sends SMS alert
   - Logs incident
