🚨 SafeHer

Discreet Safety Assistant

💡 About the Project

SafeHer is a smart personal safety application designed to help individuals in unsafe environments request help without attracting attention. It appears as a normal calculator but contains hidden emergency features.

🔐 Key Features

  • Hidden SOS trigger using secret code
  • Emergency SMS alert to trusted contact
  • GPS location sharing with Google Maps link
  • Automatic audio recording for evidence
  • Webcam photo capture
  • Incident logging with timestamps

🛠 Technologies Used

  • Python
  • Tkinter
  • OpenCV
  • Geolocation API
  • Twilio API
